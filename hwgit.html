What is Docker

Dockers makes it easier to diploy and run applications using containers. Containers allow one to to store an application and all of its parts as one package. Think of this as a contained ecosystem with everything it needs to sustain it self. Additionally, Docker is open source like other things you might be familiar with like R or Python. This allows anyone to contribute to Docker and extend it's functionality. For these reasons Dockers allows business to deploy applications faster potentially incrasing revenue or efficiency.

Source: https://opensource.com/resources/what-docker
Source: https://www.docker.com/why-docker

What is Nginx

Nginx is a tool you can use for web serving, reverse proxing, caching, load balencing, and media streaming. This may be sound familiar to digital ocean, however, there are more capabilties with Nginx. With it's open source platform the collaboration from others has really helped the development of this tool.

Source: https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

Virtual Machine acts like a computer but is a file often refered to an an image. A virtual machine is seperate from the rest of the computer making it a ideal environment for testing or working with virus infected data. Each VM has it's own hardware making it a good solution for any team to run any software or applicated not intended for every day use.

Source: https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/
